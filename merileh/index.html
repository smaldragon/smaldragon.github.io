Technical

  • The Mehrileh script is implemented in the Unicode Private Use Area from U+E000 to U+E021, which means it won't get in conflict/replace any other official unicode symbols, these characters can be copied into minecraft from the unicode.txt file inside the resource pack .zip or from this page.
