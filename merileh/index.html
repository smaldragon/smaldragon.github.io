Merileh Script

The following page contains info and links for the Merileh conlang and its script.

Technical

  • The Merileh script is implemented in the Unicode Private Use Area from U+E000 to U+E021, which means it won't get in conflict/replace any other official unicode symbols, these characters can be copy-pasted into minecraft from the unicode.txt file inside the resource pack .zip or from this page.
